Autonomous Diagnostic Language Protocol

ADLP is a structured diagnostic registry for autonomous and robotic systems. It organizes diagnostic events by domain, system, fault type, and severity.

Open Validator Browse Registry

Code Format

ADLP codes use the format: DOMAIN.SYSTEM.FAULT.SEVERITY

Example: PROPULSION.MOTOR.OVERHEAT.S3

What you can do here

Use the Validator to check whether a full ADLP code is valid against the registry.
